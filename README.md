# 🎯 Boarded

> **A comprehensive client management platform for freelancers, indie founders, and small business owners.**

Built with Next.js, Supabase, and Tailwind CSS. Manage your entire client workflow from onboarding to delivery in one beautiful workspace.

---

## ✨ Features

- 📊 **Client Management** - Organize and track all client relationships
- 📝 **Contracts & Legal Docs** - Generate agreements with digital signatures
- 💰 **Invoices & Payments** - Professional invoicing with automatic calculations
- ⏱️ **Time Tracking** - Live timer with billable hours tracking
- 📋 **Timesheets** - Clean, Clockify-style timesheet views
- 🔗 **Live Preview Sharing** - Share temporary preview links with clients
- 💬 **Project Collaboration** - Feedback threads and comments
- 📁 **File Management** - Secure file sharing with expiring links
- 🔔 **Notifications** - Smart reminders and activity tracking
- 📈 **Dashboard** - Beautiful overview of all your work

---

## 🚀 Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Set up environment variables
cp .env.local.example .env.local
# Edit .env.local with your Supabase credentials

# 3. Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) - you're ready to go! 🎉

## 🛠️ Tech Stack

- **Framework:** Next.js 14 (React 18)
- **Database:** Supabase (PostgreSQL)
- **Authentication:** Supabase Auth
- **Storage:** Supabase Storage
- **Styling:** Tailwind CSS
- **Icons:** React Icons (Feather)
- **PDF Generation:** jsPDF
- **Signatures:** react-signature-canvas
- **Dates:** date-fns
- **Animations:** Framer Motion
