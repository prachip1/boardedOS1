### 🎨 Color Usage Guide

```css
/* Primary Actions */
bg-accent → Purple-blue
bg-brand-green → Success states
bg-brand-sky → Info states
bg-brand-purple → Special highlights

/* Backgrounds */
bg-background → Pure black
bg-background-secondary → #0a0a0a
bg-background-tertiary → #111111
bg-background-elevated → #1a1a1a

/* Text */
text-text-primary → White
text-text-secondary → Gray (#9ca3af)
text-text-tertiary → Darker gray (#6b7280)

/* Borders */
border-border → #1f1f1f
border-border-hover → #2a2a2a
```
