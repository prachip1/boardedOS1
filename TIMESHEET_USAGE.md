# 📝 How to Use Timesheets

## Two Ways to Fill Timesheets

### 1. **Automatic - From Time Tracker** ⏱️

When you use the Time Tracking page:

```
1. Go to Time Tracking
2. Select project
3. Enter task name
4. Click "Start"
5. Work on your task
6. Click "Stop"
→ Entry automatically saved to timesheet!
```

**Best for:**
- Active work sessions
- Real-time tracking
- Accurate time logging

---

### 2. **Manual - Add Entry** ✍️

When you need to add past work:

```
1. Go to Timesheets
2. Click "Add Entry" button
3. Fill in the form:
   - Date
   - Client
   - Project
   - Task description
   - Start & end time
   - Notes (optional)
   - Billable checkbox
   - Private checkbox
4. Click "Save Entry"
→ Entry added to timesheet!
```

**Best for:**
- Forgot to track time
- Adding yesterday's work
- Batch entry at end of day
- Manual corrections

---

## 📋 Add Entry Form

### Fields:

**Required:**
- **Date** - When you worked
- **Client** - Which client
- **Project** - Project name
- **Task** - What you did
- **Start Time** - When you started (e.g., 9:00 AM)
- **End Time** - When you finished (e.g., 11:00 AM)

**Optional:**
- **Notes** - Additional details
- **Billable** ✓ - Is this billable to client?
- **Private** ✓ - Hide from client when sharing?

### Example:
```
Date: Oct 18, 2025
Client: Acme Corporation
Project: Website Redesign
Task: Implemented responsive navigation menu
Start: 9:00 AM
End: 11:30 AM
Notes: Added mobile dropdown and smooth animations
☑ Billable
☐ Private
```

## 🔄 Workflow Options

### Option A: Track as you go
```
Morning:
1. Start timer for Task A
2. Work for 2 hours
3. Stop timer → Auto-saved

Afternoon:
1. Start timer for Task B
2. Work for 1.5 hours
3. Stop timer → Auto-saved

End of day: Timesheet complete! ✅
```

### Option B: Fill at end of day
```
End of day:
1. Go to Timesheets
2. Click "Add Entry" for each task
3. Enter: 9-11 AM, Task A
4. Enter: 2-3:30 PM, Task B
5. Enter: 4-5 PM, Task C

Timesheet complete! ✅
```

### Option C: Mix both
```
- Use timer for long tasks
- Manually add short tasks
- Add forgotten items later
```

## 💡 Best Practices

### ✅ DO:
- **Be specific** - "Built login form" not "Coding"
- **Track daily** - Don't wait until Friday
- **Use categories** - Mark billable vs non-billable
- **Add notes** - Future you will thank you
- **Mark private** - Hide internal meetings

### ❌ DON'T:
- Forget to track time
- Use vague descriptions
- Let entries pile up
- Track in 5-minute increments (round to 15 min)
- Forget to mark privacy

## 🎯 Common Scenarios

### Scenario 1: Forgot to track time
```
Problem: Worked 2 hours, forgot to start timer
Solution: 
1. Go to Timesheets
2. Click "Add Entry"
3. Enter time manually
4. Save
```

### Scenario 2: Timer ran too long
```
Problem: Left timer running during lunch
Solution:
1. Stop timer
2. Go to Timesheets
3. Edit entry (adjust end time)
4. Save
```

### Scenario 3: Multiple small tasks
```
Problem: Did 5 small tasks (10 min each)
Solution:
1. Either: Track each separately
2. Or: Combine into one entry "Various bug fixes - 50 min"
```

### Scenario 4: Client meeting
```
Problem: 30-min meeting, not billable
Solution:
1. Add entry manually
2. ☐ Uncheck "Billable"
3. ☑ Check "Private"
4. Won't show to client
```

## 📊 Quick Reference

### Keyboard Workflow:
```
Time Tracking page:
- Select project → Tab
- Enter task → Tab
- Click Start → Work → Click Stop

Timesheet page:
- Click "Add Entry"
- Fill form top to bottom
- Tab through fields
- Save
```

### Time Format:
```
Use 12-hour format:
✅ 9:00 AM, 2:30 PM
✅ 09:00, 14:30 (24-hour also works)
```

### Duration:
```
Calculated automatically:
9:00 AM - 11:30 AM = 2:30 (2 hours 30 min)
```
